# Hello Happy Frontend

Frontend React do projeto Hello Happy, criado com Vite.

## Scripts

Rode os comandos a partir da pasta raiz do projeto:

```bash
npm --prefix frontend install
npm --prefix frontend run dev
```

Build de producao:

```bash
npm --prefix frontend run build
```

Lint:

```bash
npm --prefix frontend run lint
```

Durante o desenvolvimento, o Vite atende em http://localhost:5173 e usa proxy para chamar o backend pelo caminho `/api`.
