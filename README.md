# Hello Happy

E-commerce de brinquedos desenvolvido com Spring Boot.

## Tecnologias

- Java 21
- Spring Boot 3.2.3
- Spring Data JPA / Hibernate
- PostgreSQL
- Thymeleaf
- Maven

## Requisitos

- JDK 17+
- PostgreSQL com database `hello_happy` criado

## Como executar

```bash
./mvnw spring-boot:run
```
